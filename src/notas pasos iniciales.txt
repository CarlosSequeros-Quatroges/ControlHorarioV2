PASOS PREVIOS
- crear un repositorio vacio en github
- clonar repositorio en carpeta proyectos (crea la carpeta con el nombre del repositorio)
- desde terminal en carpeta proyecto crear base angular (ng install [nombre proyecto] --directory [carpeta   .\])
- ng serve

-(si no se ha creado, copiar un favico en src)
